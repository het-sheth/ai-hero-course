## Build A Feature (03.04)

- [[day-1-fundamentals/build-a-feature|Build A Feature]] — first feature: a 1–5 star course-rating system (no written reviews), shown on the list page and course page. The point wasn't the feature but the **workflow**: lightweight 1–2 sentence prompt, then observe Claude's default behavior (Explore subagent? what files? what questions?), review its plan and steer before implementing, flip to accept-all-edits (shift+tab), and stay context-paranoid — `/context` repeatedly, nervous around the **40% (~80K token)** dumb-zone line.

## Non-Determinism (03.05)

- [[day-1-fundamentals/non-determinism|Non-Determinism]] — a detour setting expectations: agents are **next-token-chooser machines**, so the same prompt on the same codebase can give different answers. Responses sit on a probability curve (most sensible, occasional weird outliers). It's baked in, not a setup bug — "ride the wave," and you can make agents more consistent later via AFK feedback loops.
