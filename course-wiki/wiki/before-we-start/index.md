# Before We Start

* [Navigating The Discord](/before-we-start/navigating-the-discord.md) - The cohort's main support channel — community vs cohort-only channels, where to ask, and the hero system.
* [Repo Setup](/before-we-start/repo-setup.md) - Getting the course playground running locally: clone the repo, install Node + pnpm, seed the database, run the dev server, and connect an agent.
* [Database Migrations](/before-we-start/database-migrations.md) - Keeping the database in sync with your code via a three-step Drizzle workflow: edit the schema, generate migration files, apply them. And how to reset when things go wrong.
* [Exercise Workflow](/before-we-start/exercise-workflow.md) - The course is interactive exercises in the playground — to do one yourself you get your repo into the same state it was when the exercise was recorded. Three commands handle this: pnpm reset, pnpm cherry-pick, and pnpm pull.
* [Setting Up Claude For The Course](/before-we-start/claude-setup.md) - Picking the agent, model, effort level, and subscription tier for the cohort. Short version: use Claude Code on your tier's default model at medium effort.
* [Office Hours](/before-we-start/office-hours.md) - Live YouTube office hours — three sessions, optional, all recorded as VODs.
* [Appendix: Cohort 003 → 004 Changelog](/before-we-start/cohort-003-to-004-changelog.md) - Diff of 003 → 004. Key decoder: 'Ralph' = AFK Agents.
