A diff of what changed from Cohort 003 → 004. Only relevant if you took 003 — but the terminology
note below is useful for decoding older community/Discord references.

> [!TIP] "Ralph" = AFK Agents
> Day 5 was **renamed from "Ralph" to "AFK Agents"**, and every "Ralph" lesson was renamed to use
> "AFK Agent". So when older community or Discord posts mention **Ralph**, they mean today's
> [[day-5-afk-agents/afk-agents|AFK agents]].

## What changed, by module

| Module | Changes |
|--------|---------|
| **Before We Start** | New: Database Migrations |
| **Getting To Know Claude Code** | New: Intro |
| **Day 1 · Fundamentals** | New: Non Determinism · New: Handing Off · Renamed: *What Is Plan Mode* → *Why Plan Mode Sucks* · Renamed: *The Plan Execute Clear Loop* → *The Grill Execute Clear Loop* |
| **Day 2 · Steering** | Renamed: *A Skill For Writing Skills* → *Using Skills For Steering* |
| **Day 3 · Planning** | New: Ask User Question · Removed: The Grill Me Skill |
| **Day 4 · Feedback Loops** | No changes |
| **Day 5 · AFK Agents** | Section renamed Ralph → AFK Agents. Removed: *HITL vs AFK Ralph*. Renamed: *What Is Ralph* → *What Is An AFK Agent*; *Trying HITL Ralph* → *Trying HITL Agents*; *Setting Up And Trying AFK Ralph* → *Setting Up And Trying AFK Agents*; *Using Backlogs To Queue Tasks For Ralph* → *…For AFK Agents*; *Hooking Up Ralph To Your Backlog* → *Hooking Up Agents To Your Backlog* |
| **Day 6 · HITL Patterns** | New: The Prototype Skill · New: The Final Workflow |

> [!NOTE] Naming note
> The source page notes that Matt's draft called the last new Day 6 item "Wrapping Up"; in the
> current Cohort 004 resource map that lesson is titled **The Final Workflow**.
