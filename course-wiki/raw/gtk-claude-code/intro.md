## Lesson page content

In this course, Matt primarily uses **Claude Code** as the agent for driving the lessons,
demonstrating things, and showing how agents in general work — because it's a popular harness you
may have used, and the one he uses most right now. However, **the course is harness-agnostic** —
you can choose whatever harness you like, as long as it runs in a CLI.

For folks who've never used a coding harness, or who just want a bit more out of Claude Code, this
primer section gives the basics before getting started. If you haven't installed Claude Code yet,
the setup instructions are in the official docs. And if you're experienced with Claude Code, you'll
probably still pick up a few tips — it's not just the basics; there are a couple of advanced things
in there too.
